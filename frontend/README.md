# Physical AI & Humanoid Robotics Book - Frontend

This is the frontend for the interactive Physical AI and Humanoid Robotics book, built with Docusaurus.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## Deployment

The site is deployed to GitHub Pages. To deploy:

```bash
npm run deploy
```

## Features

- Interactive book content using Docusaurus
- Dark blue gradient theme with purple accents
- Glassmorphism design elements
- Responsive design for all devices
- Integration with backend API for RAG chatbot, personalization, and translation
- Support for multiple languages (Urdu translation)

## Tech Stack

- Docusaurus v3+
- React
- Custom CSS with glassmorphism effects
- Integration with FastAPI backend
