# مبارک ہو!

آپ نے ابھی **ڈوکوسورس کی بنیادی باتیں** سیکھی ہیں اور **ابتدائی ٹیمپلیٹ** میں کچھ تبدیلیاں کی ہیں۔

ڈوکوسورس کے پاس **پیش کرنے کے لیے بہت کچھ ہے**!

کیا آپ کے پاس **5 اور منٹ ہیں**؟ **[ورژن کنٹرول](../tutorial-extras/manage-docs-versions.md)** اور **[i18n](../tutorial-extras/translate-your-site.md)** کو دیکھیں۔

کیا اس ٹیوٹوریل میں کچھ **غیر واضح** یا **بگ** ہے؟ [براہ کرم اس کی اطلاع دیں!](https://github.com/facebook/docusaurus/discussions/4610)

## اگلا کیا ہے؟

- [سرکاری دستاویزات](https://docusaurus.io/) پڑھیں
- [`docusaurus.config.js`](https://docusaurus.io/docs/api/docusaurus-config) کے ساتھ اپنی سائٹ کی ترتیب میں ترمیم کریں
- [`themeConfig`](https://docusaurus.io/docs/api/themes/configuration) کے ساتھ نیویگیشن بار اور فوٹر آئٹمز شامل کریں
- ایک کسٹم [ڈیزائن اور لے آؤٹ](https://docusaurus.io/docs/styling-layout) شامل کریں
- ایک [تلاش بار](https://docusaurus.io/docs/search) شامل کریں
- [ڈوکوسورس شوکیس](https://docusaurus.io/showcase) میں حوصلہ افزا تلاش کریں
- [ڈوکوسورس کمیونٹی](https://docusaurus.io/community/support) میں شامل ہوں
